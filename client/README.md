## Overview

**project-management** is a lightweight, open-source workspace management tool that lets teams organize projects, track tasks, and collaborate — all in one place. Built with ReactJS and styled with Tailwind CSS, it's fast, extensible, and easy to self-host.

---

## Features

| Feature                | Description                                                                    |
| ---------------------- | ------------------------------------------------------------------------------ |
| 🗂️ Multiple Workspaces | Create isolated workspaces, each with its own projects, tasks, and members     |
| 📋 Project Management  | Manage projects end-to-end with full visibility over team members and progress |
| 📊 Analytics           | View completion rates, project progress, and team size at a glance             |
| ✅ Task Management     | Assign tasks, set due dates, and track status in real time                     |
| 👥 User Management     | Invite members, manage roles, and monitor user activity                        |

## Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or later recommended)
- npm, yarn, pnpm, or bun

### Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/GreatStackDev/project-management.git
cd project-management
npm install
```

### Running the App

Start the development server:

```bash
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

> The page auto-updates as you edit files. Start with `src/App.jsx`.
